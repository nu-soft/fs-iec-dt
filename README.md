# F# IEC 61131-3 data types
ATM library only adds IEC 61131-3 data type aliases to F#
aliases as follows:

| BYTE  | uint8   |
|-------|---------|
| WORD  | uint16  |
| DWORD | uint32  |
| LWORD | uint64  |
| SINT  | int8    |
| USINT | uint8   |
| INT   | int16   |
| UINT  | uint16  |
| DINT  | int32   |
| UDINT | uint32  |
| LINT  | int64   |
| ULINT | uint64  |
| REAL  | float32 |
| LREAL | float   |
